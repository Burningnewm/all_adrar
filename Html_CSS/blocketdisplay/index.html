DISPLAY BLOCK ET POSITIONNEMENT

DISPLAY BLOCK

Utilisez la propriété display:block ; pour transformer les liens en bloc (pour pouvoir leur donner des dimensions). Donnez à vos liens les dimensions exactes de votre image de fond. Utilisez la propriété line-height: px avec une valeur identique à la hauteur du block afin que le texte soit centré verticalement.
